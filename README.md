# Java Task Manager

A simple console task manager built in Java.

## Features

- Add tasks
- List tasks
- Mark tasks as completed
- Remove tasks

## Technologies

- Java
- ArrayList
- Object-Oriented Programming

## How to run

Clone the repository:

git clone https://github.com/luishannisch/java-task-manager.git

Compile:

javac src/*.java

Run:

java src.Main
